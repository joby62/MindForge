## 当前规则

- 根目录不再新增新的业务文档
- 当前 canonical 文档已经迁入 `docs/`
- 根目录不再作为业务文档的正式存放位置
- 新增文档统一进入 `docs/`
- 会议记录、闸门审查、打回记录、决策记录，不再散落，统一进入 `docs/03_records/`
- 临时草稿只能进 `docs/98_inbox/`

## 目录说明

- `docs/00_governance/`：文档治理规则
- `docs/01_registry/`：当前文档登记册与迁移清单
- `docs/02_templates/`：统一模板
- `docs/03_records/`：闸门审查、工作日志、决策记录
- `docs/10_truth/`：唯一真相与总纲
- `docs/20_strategy/`：战略、边界、世界观上游
- `docs/30_workflows/`：流程、RACI、模板、SOP
- `docs/40_lessons/`：按 lesson 归档的规格文档
- `docs/50_execution/`：执行包、工程输入包
- `docs/60_reference/`：仍有参考价值但不再作为当前唯一真相的文档
- `docs/98_inbox/`：临时草稿
- `docs/99_archive/`：归档
