# lesson_01 / cards

这里是 `lesson_01` 的 8 张卡正文区。

规则：

1. 每位元老只编辑自己负责的那一张卡
2. 不允许直接改其他卡正文
3. 发现问题，只能在自己的卡里标风险、建议打回或上报
4. 每张卡写完后都要补交棒块

文件对应关系：

1. `01_goal_card.md` -> 节目标官
2. `02_hook_card.md` -> 戏剧钩子官
3. `03_system_card.md` -> 系统建模官
4. `04_error_card.md` -> 错误反馈官
5. `05_interaction_card.md` -> 交互路径官
6. `06_cognition_card.md` -> 认知校验官
7. `07_parent_evidence_card.md` -> 家长证据官
8. `08_template_archive_card.md` -> 模板沉淀官
