# MVP 世界观主设定

版本：v0.2  
状态：第一章第一节聚焦版  
适用范围：希腊神话 MVP

## 1. 当前唯一世界句

`孩子和自己的小精灵搭档，一起进入被局部拨动的神话系统现场，通过对话理解异常并修补它。`

## 2. 已锁定事实

- `Vibe Coding for Kids` 已升级为一级战略
- 孩子世界里有一个固定搭档 NPC
- 这个固定搭档当前形态是：会飞的小精灵
- 孩子主要通过和这位小精灵对话来推进任务
- 神话角色不承担主教学职责
- 神话世界在正常状态下应当逻辑自洽
- 异常来自扰动制造者对局部变量的拨动
- 第一章扰动制造者类型已锁定为：条件拨针者
- 第一章扰动制造者临时代号已锁定为：灰烬拨针者
- 第一节从“普罗米修斯盗火之后，人间初火系统失稳”开始
- 第一节正式节型已锁定为：修补裂缝节
- 第一节正式系统对象已锁定为：初火祭坛稳焰系统

## 3. 角色职责边界

### 小精灵

- 气质锁定为：机灵侦察型
- 必须可信赖，会追问，会抓异常点
- 可以帮孩子缩小范围，但绝不替孩子下结论
- 它是孩子在第一节里的主对话对象

### 神话角色

- 负责提供世界重量、情绪张力和现场反馈
- 可以有存在感，但不能抢走主对话权
- 不写成老师、客服或任务引导员

### 灰烬拨针者

- 不是传统大反派，而是局部变量拨动者
- 第一节里的职责是把一个原本应稳住的条件拨到临界状态
- 它制造的是“半成功异常”，不是全面崩坏

## 4. 第一节现场边界

第一节只需要一个足够清楚的异常现场：

- 火已经被带到人间
- 第一座火坛已经亮起
- 但火焰始终稳不住

这意味着第一节不需要：

- 大地图探索
- 多个神话地点切换
- 大灾难级世界崩坏
- 复杂通道 lore

## 5. 本轮只交四个产物

1. 小精灵设定卡
2. 灰烬拨针者设定卡
3. 第一节开场钩子草案
4. 世界语言与禁用表达

四个产物的统一正文收录在 [lesson_01_brief.md](/Users/lijiabo/MindForge/docs/40_lessons/lesson_01/lesson_01_brief.md)。

## 6. 本轮暂不拍板

- 小精灵的正式名字
- 小精灵的具体外观资产
- 灰烬拨针者的完整身份背景
- 第一节具体被拨动的是哪一个稳焰条件
