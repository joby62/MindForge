# 当前阶段唯一真相

版本：v0.2  
状态：已拍板

## 阶段唯一真相

`我们现在要先证明：孩子能和固定搭档小精灵对话，完成一节真实的系统修补闭环，并让家长清楚看见这种思维成长。`

## 这轮已锁定的事实

- `Vibe Coding for Kids` 已升级为一级战略
- 当前样板只看：希腊神话 MVP / 第一章 / 第一节
- 孩子主要通过和固定搭档小精灵对话推进任务
- 神话角色可以有重量和戏剧感，但不承担主教学职责
- 第一章扰动制造者是“条件拨针者”，临时代号“灰烬拨针者”
- 第一节从“普罗米修斯盗火之后，人间初火系统失稳”开始
- 第一节类型是“修补裂缝节”
- 第一节系统对象是“初火祭坛稳焰系统”

## 这轮只交四个产物

1. 小精灵设定卡
2. 灰烬拨针者设定卡
3. 第一节开场钩子草案
4. 世界语言与禁用表达

四个产物的统一正文收录在 [lesson_01_brief.md](/Users/lijiabo/MindForge/docs/40_lessons/lesson_01/lesson_01_brief.md)。

## 这轮明确不做

- 不扩写整套世界观
- 不扩章节、季结构和总反派线
- 不补宏大 lore
- 不让 AI / NPC 替孩子做核心判断
- 不把对话做成聊天皮肤

## 当前判断标准

这一轮是否成立，只看三件事：

1. 孩子是否愿意在 2 分钟内继续往下走
2. 孩子的第一次判断是否通过和小精灵的对话发生
3. 神话重量是否存在，但没有抢走主对话权
