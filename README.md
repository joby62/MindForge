# MindForge

MindForge is the working repository for the Task Adventure Engine / Vibe Coding for Kids documentation system.

## Start Here

The canonical document entrypoint is [docs/README.md](/Users/lijiabo/MindForge/docs/README.md).

From there you can find:

- current single truth
- master outline
- workflow and template docs
- lesson-specific specs
- execution packets
- gate reviews, worklogs, and decision logs

## Structure

- `docs/10_truth/`: current truth and master outline
- `docs/20_strategy/`: strategy and world boundary docs
- `docs/30_workflows/`: workflows, templates, RACI
- `docs/40_lessons/`: lesson-specific specs
- `docs/50_execution/`: execution plans and Codex packets
- `docs/03_records/`: gate reviews, decision logs, worklogs
- `archive/`: archived source materials

## Current Rule

Business documentation should be added under `docs/`, not at the repository root.
